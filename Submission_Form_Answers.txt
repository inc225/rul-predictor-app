Project Title:
Predictive Maintenance for Aircraft Engines: A Machine Learning Approach to Remaining Useful Life (RUL) Prediction

Project Abstract:
This project develops a supervised machine learning framework to estimate the Remaining Useful Life (RUL) of aircraft turbofan engines using sensor data from NASA's C-MAPSS FD001 degradation dataset. Predictive maintenance is important in aviation because it can reduce unplanned downtime, improve safety, and support more efficient maintenance scheduling. The workflow preprocesses multivariate engine time-series data, creates capped RUL labels, removes low-information features, and compares Ridge Regression, Random Forest, and Gradient Boosting models. The Random Forest model achieved the best performance on the FD001 test set, with an RMSE of 18.20 cycles and MAE of 13.18 cycles on capped RUL labels. A Streamlit web application was also developed to allow users to upload engine sensor data or manually enter measurements and receive an estimated RUL prediction with a maintenance-priority interpretation.
